SimpleOverlayLogStorageService

SimpleOverlayLogStorageService class

Service for storing logs to persistent storage.

This service uses an isolate to write logs to the application support directory. The logs are written in JSON Lines format.

The logs are written to two files, simple_logs.jsonl and network_logs.jsonl. The first file contains logs from the LoggerController.log method, while the second file contains logs from the NetworkLoggerInterceptor.
